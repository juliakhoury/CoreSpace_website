# CORESPACE

CoreSace is a collabarative game that encourages players to stay productive,
the game has some productive tools that are most needed in a group study session 
and the most inportant tool is an AI chat that is build specifically to assist studying.

🔗 Live Site (https://juliakhoury.github.io/CoreSpace_website/)

---

## About the Project

The project was inspired by a website called gather which is a virtual workspace.
We wanted to make something smimilar but tailor it towards students 
and study groups and add have a story in the background of the game.
Most notiable productivity tool we wanted to include is the AI chat that ran locally on each device.
